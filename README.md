# Ferramenta-para-organizar-declaracao-de-imposto-de-renda
Contém o arquivo do organizador de declaração de Imposto de Renda criado em atividade do curso de Excel com IA na plataforma DIO. 
A ferramenta organiza as informações que o contribuinte deve apresentar anualmente à receita federal, facilitando assim o processo de declaração de renda.
É uma ferramente útil tanto para pessoas físicas que optem por realizar sua declaração de IR de forma autônoma, quanto para profissionais de contabilidade que trabalham com muitos clientes.
